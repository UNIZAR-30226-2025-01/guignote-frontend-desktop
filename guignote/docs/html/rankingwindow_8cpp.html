Detailed Description

Implementación de la clase RankingWindow.

Este archivo forma parte del Proyecto de Software 2024/2025 del Grado en Ingeniería Informática en la Universidad de Zaragoza.

Este archivo define la ventana de rankings, que muestra listas de puntuaciones individuales o por parejas, con opción de filtrar solo amigos. Se encarga de la UI, de obtener datos del servidor y de actualizar la lista.

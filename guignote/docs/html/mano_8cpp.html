Detailed Description

Implementación de la clase Mano.

Este archivo forma parte del Proyecto de Software 2024/2025 del Grado en Ingeniería Informática en la Universidad de Zaragoza.

Define la clase Mano, que gestiona un conjunto de cartas para cada jugador, posicionándolas automáticamente según la orientación (inferior, superior, izquierda o derecha) y permite añadir o eliminar cartas.

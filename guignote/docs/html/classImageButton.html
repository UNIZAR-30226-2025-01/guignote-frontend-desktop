Detailed Description

Botón de imagen con texto y efectos interactivos.

La clase ImageButton hereda de QLabel y permite mostrar una imagen junto a un texto. Además, implementa efectos interactivos (como oscurecer la imagen) al pasar el ratón o hacer clic, emitiendo una señal para notificar la interacción.

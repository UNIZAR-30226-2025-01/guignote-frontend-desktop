Detailed Description

Implementación de la clase SettingsWindow.

Este archivo forma parte del Proyecto de Software 2024/2025 del Grado en Ingeniería Informática en la Universidad de Zaragoza.

Esta clase define la ventana de configuración de la aplicación, permitiendo al usuario modificar opciones de visualización, como cambiar entre el modo ventana y el modo pantalla completa. La interfaz se compone de un sidebar para la navegación y un área (QStackedWidget) para mostrar la página de "Gráficos", que incluye opciones configurables mediante botones de radio y un botón de cierre.

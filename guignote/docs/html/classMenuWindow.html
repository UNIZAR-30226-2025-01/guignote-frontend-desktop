Detailed Description

Ventana principal del menú de la aplicación.

Gestiona la interfaz que permite al usuario navegar entre modos de juego, acceder a la configuración, ver la lista de amigos, rankings e inventario. También reproduce música de fondo y mantiene la conexión WebSocket para emparejamientos y reconexiones de partidas.

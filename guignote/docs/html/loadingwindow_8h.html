Detailed Description

Declaración de la clase LoadingWindow, pantalla de carga de la aplicación.

Este archivo forma parte del Proyecto de Software 2024/2025 del Grado en Ingeniería Informática en la Universidad de Zaragoza.

Esta clase representa la pantalla de carga inicial que muestra una animación GIF ("carga.gif") a pantalla completa durante unos segundos, seguido de un efecto de desvanecimiento que da paso a la ventana principal del menú (MenuWindow).

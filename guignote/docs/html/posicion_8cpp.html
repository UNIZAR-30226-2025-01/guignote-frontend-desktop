Implementación de la clase Posicion.

Este archivo forma parte del Proyecto de Software 2024/2025 del Grado en Ingeniería Informática en la Universidad de Zaragoza.

Este archivo define la clase Posicion, responsable de representar cada zona de juego donde puede caer una carta. Gestiona su posición, rotación, drag&drop y notifica al servidor cuando se juega una carta.

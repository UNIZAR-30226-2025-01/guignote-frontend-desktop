Detailed Description

Implementación de la clase RegisterWindow.

Este archivo forma parte del Proyecto de Software 2024/2025 del Grado en Ingeniería Informática en la Universidad de Zaragoza.

Esta clase representa la ventana de registro de la aplicación. Permite a los usuarios crear una nueva cuenta proporcionando sus credenciales, y contiene la lógica para enviar los datos al servidor mediante una petición HTTP POST. Además, ofrece la opción de cambiar a la ventana de inicio de sesión a través de una señal.

Detailed Description

Declaración de la clase Mano, que representa la mano de un jugador.

Este archivo forma parte del Proyecto de Software 2024/2025 del Grado en Ingeniería Informática en la Universidad de Zaragoza.

La clase Mano gestiona las cartas que posee un jugador durante la partida. Permite añadir, eliminar y mostrar cartas, y mantiene información sobre la posición y el jugador asociado.
